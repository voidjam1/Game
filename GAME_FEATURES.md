# 镜界电梯 - 视觉小说游戏

## 最新更新内容

### ✨ 新增功能

1. **逐字显示动态效果** 
   - 文字会逐个显示，营造打字机效果
   - 可以通过设置调整文字显示速度（1-10级）
   - 点击对话框可快速跳过显示完整文本

2. **自动播放功能**
   - 按 `A` 键或从菜单中启用自动播放
   - 文字显示完毕后会自动停顿0.5秒，然后播放下一句
   - 可在设置中调整自动播放速度

3. **对话历史记录**
   - 按 `H` 键或从菜单中打开对话历史
   - 显示所有已浏览的对话内容
   - 包含时间戳和角色名称

4. **小头像系统**
   - 主角头像显示在对话框左侧
   - 配角头像显示在对话框右侧
   - 说话时对应头像会亮起，另一个变暗
   - 用亮圈标识当前说话的角色

5. **章节系统**
   - 剧本分为【序章】和【EP1】等章节
   - 存档界面会显示该存档所属的章节
   - 便于玩家快速定位游戏进度

6. **成就系统**
   - 完成特定剧情会解锁成就
   - 成就解锁时会显示动画通知
   - 可从菜单或主菜单查看所有成就
   - 当前已添加的成就：
     - 🌟 初次觉醒 - 第一次睁开眼睛
     - 🔍 调查员 - 完成所有初始调查
     - 🚪 踏入未知 - 进入电梯
     - 🪞 镜之试炼 - 通过第9层
     - 📞 无名之人 - 通过第8层
     - 👻 镜中的死神 - 达成真结局

7. **剧本分句显示**
   - 长对话已拆分成多个节点
   - 节奏更加流畅自然

8. **标题画面背景图支持**
   - 可在 `App.tsx` 中设置 `backgroundImage` 属性
   - 支持自定义主菜单背景图

## 🎮 操作指南

### 键盘快捷键
- `空格` / `回车` - 继续对话
- `Ctrl + S` - 快速保存
- `← 左箭头` - 回退上一句
- `A` - 切换自动播放
- `H` - 打开对话历史
- `ESC` / `菜单按钮` - 打开游戏菜单

### 游戏菜单功能
- 快速保存 - 保存到快速存档位
- 保存 - 选择存档槽位保存
- 读取 - 加载之前的存档
- 回退 - 返回上一个对话节点
- 对话历史 - 查看所有对话记录
- 自动播放 - 开启/关闭自动播放
- 画廊 - 查看已解锁的CG
- 成就 - 查看成就进度
- 设置 - 调整游戏设置

## 📝 添加内容指南

### 1. 添加人物头像

在 `src/app/data/story.ts` 中的角色定义里添加 `avatar` 字段：

```typescript
export const characters: Record<string, Character> = {
  wanhui: {
    id: 'wanhui',
    name: '万辉',
    sprite: '立绘图片URL',
    avatar: '头像图片URL', // 添加这一行
  },
};
```

### 2. 添加背景图/CG

直接发送图片给我，我会：
1. 将图片上传到合适的位置
2. 在剧本中对应节点添加 `background` 或 `cg` 字段
3. 更新相关代码

### 3. 添加新章节

1. 在 `story.ts` 中的 `chapters` 对象里添加：
```typescript
export const chapters = {
  prologue: '序章',
  ep1: 'EP1 - 下降',
  ep2: 'EP2 - 新章节名', // 添加新章节
};
```

2. 在剧本节点中添加 `chapter` 字段：
```typescript
{
  id: 'new_node',
  type: 'dialogue',
  chapter: 'ep2', // 指定章节
  text: '对话内容',
  next: 'next_node',
}
```

### 4. 添加成就

1. 在 `story.ts` 中的 `achievements` 对象里添加：
```typescript
export const achievements: Record<string, Achievement> = {
  new_achievement: {
    id: 'new_achievement',
    title: '成就标题',
    description: '成就描述',
    icon: '🎉', // 可选的emoji图标
  },
};
```

2. 在剧本节点中添加 `achievement` 字段：
```typescript
{
  id: 'important_node',
  type: 'dialogue',
  text: '重要剧情',
  achievement: 'new_achievement', // 触发成就
  next: 'next_node',
}
```

### 5. 设置标题画面背景

在 `src/app/App.tsx` 中找到 `MainMenu` 组件，取消注释并设置背景图：

```typescript
<MainMenu
  // ...其他props
  backgroundImage="你的背景图URL"
/>
```

## 🔧 当前配置

### 文字速度设置
- 默认速度：5（中等）
- 可调范围：1-10
- 速度越高，文字显示越快

### 自动播放设置
- 默认速度：5（中等）
- 可调范围：1-10
- 停顿时间：0.5秒（固定）

### 存档系统
- 最大存档数：10个
- 额外1个快速存档位（自动）
- 存档包含：进度、章节、解锁内容、成就
